# ポモドーロタイマー

HTML、CSS、JavaScriptで作成したポモドーロタイマーアプリです。

## デモ

https://strangehumen.github.io/pomodoro/

## 制作背景

集中して作業する時間と休憩時間を管理するため、自分用のポモドーロタイマーとして制作しました。

作業時間や休憩時間を自由に変更できるようにし、スマートフォンでも使いやすいデザインを目指しました。

## 主な機能

- 作業時間のカウントダウン
- 休憩時間のカウントダウン
- スタート
- 一時停止
- リセット
- スキップ
- 自動モード切り替え
- 作業時間の変更
- 休憩時間の変更
- 完了セッション数の表示
- 円形プログレスバー
- ダークモード
- レスポンシブデザイン

## 使用技術

- HTML
- CSS
- JavaScript

## 特徴

- 作業時間と休憩時間を自由に設定可能
- 円形プログレスバーによる視覚的な進捗表示
- シンプルで使いやすいUI
- スマートフォンでも快適に利用可能
